Important FAQ

what is the document object model?

The Document Object Model (DOM) is a tree-like structure that represents the elements of a web document, allowing developers to dynamically manipulate the content and structure using JavaScript. It enables interactive features on web pages.

how do you select an element from the DOM?

1. document.getElementById('myId')

2. document.getElementsByClassName('myClass')

3. document.getElementsByTagName('div')

4. document.querySelector('.myClass')

5. document.querySelectorAll('.myClass')

what is the event delegation in the context of the DOM and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element to manage events for its child elements.

Benefits:

1. Performance: Reduces memory usage.

2. Dynamic Content: Automatically applies to new child elements.

3. Simpler Code: Less code to maintain.
